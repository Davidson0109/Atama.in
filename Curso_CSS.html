APRENDA CSS

O que é:

CSS, é basicamente a forma que se estiliza do seu jeito, os elementos escritos em uma linguagem de marcação como o html.

Linguagem de marcação são sinais e códigos que definem o formato e como o texto irá exibir.

Os códigos que você usa no CSS, são referentes ao que você deseja estilizar na sua página.



Seletores:


Seletores são padrões que ajudam a identificar e selecionar elementos HTML para aplicar estilos. Eles formam a base do CSS, permitindo que você alvo selecione precisamente quais partes do seu documento HTML você deseja estilizar.



Tipos de seletores:

  • Seletores de Tipo: Selecionam elementos com base em seus tipos. Exemplo: p { color: blue; }.

  • Seletores de Classe: Selecionam elementos com base nas classes atribuídas. Exemplo: .destaque { font-weight: bold; }.

  • Seletores de ID: Selecionam elementos com base em seus IDs únicos. Exemplo: #cabecalho { font-size: 24px; }.

  • Seletores de Atributo: Selecionam elementos com base em atributos específicos. Exemplo: input[type="text"] { width: 200px; }.

  • Seletores de Descendência: Selecionam elementos que são descendentes de outros elementos. Exemplo: article p { margin-bottom: 10px; }.

  • Pseudo-classes e Pseudo-elementos: Selecionam elementos com base em estados específicos ou partes específicas do conteúdo. Exemplo: a:hover { color: red; }.

    Propriedades e valores:


    As propriedades são atributos individuais que descrevem características estilísticas, enquanto os valores especificam como essas características devem ser apresentadas. Juntos, eles formam as regras de estilo que definem a aparência dos elementos HTML em uma página web.



    Propriedades comuns:

  • color: Define a cor do texto.
  • font-size: Determina o tamanho da fonte.
  • margin: Controla as margens ao redor de um elemento.
  • padding: Define o espaço interno entre a borda e o conteúdo de um elemento.
  • border: Especifica a largura, estilo e cor da borda de um elemento.
  • background: Define a cor ou imagem de fundo de um elemento.
  • width e height: Especificam a largura e altura de um elemento.



    • Unidades:

  • px (pixels): Unidade fixa de medida.
  • em: Relativa ao tamanho da fonte do elemento.
  • %: Relativa ao elemento pai.
  • vw (viewport width): Relativa à largura da viewport.



    • Valores:

  • Cores: Podem ser especificadas por nomes, códigos hexadecimais, RGB, entre outros.
  • Fontes: Além do tamanho, você pode definir o estilo (negrito, itálico), a família da fonte, etc.
  • Posicionamento: Para propriedades como position, os valores podem ser static, relative, absolute, etc.
  • Transparência: A propriedade opacity aceita valores de 0 a 1.

    Box Model:


    O modelo de caixa (Box Model) é uma representação visual dos elementos HTML como caixas retangulares, cada uma composta por conteúdo, padding, border e margin. Esses componentes determinam o tamanho total ocupado pelo elemento na página.



    Componentes do Box Model:

  • Conteúdo (Content): É a área onde o conteúdo do elemento é exibido (por exemplo, o texto de um parágrafo).
  • Padding: Espaço transparente ao redor do conteúdo, dentro da borda.
  • Border: A borda que envolve o conteúdo e o padding.
  • Margin: A área transparente fora da borda, que cria espaço entre elementos adjacentes.


    • Propriedades relaciondas:

  • width e height: Determinam a largura e altura do conteúdo da caixa.
  • padding: Define o espaçamento entre o conteúdo e a borda.
  • border: Especifica a largura, estilo e cor da borda.
  • margin: Define o espaçamento entre a borda e elementos adjacentes.

    Layout:


    O layout em CSS refere-se à organização e posicionamento de elementos na página. O modo como os elementos são dispostos afeta a estrutura e a aparência geral do site. Existem várias técnicas para controlar o layout, permitindo designs flexíveis e responsivos.


    Propriedades de Layout:

  • display: Controla o modelo de layout usado para um elemento. Exemplos incluem block, inline, flex, e grid.
  • position: Define o método de posicionamento de um elemento em relação ao seu contêiner ou à viewport.
  • float: Move um elemento para a esquerda ou direita, permitindo que outros elementos fluam ao seu redor.


    • Layout responsivo:

  • Media queries são usadas para criar layouts responsivos, ajustando estilos com base nas características da viewport, como largura de tela.


    • Layout flexivel:

  • O modelo de layout flexível (display: flex;) permite a criação de layouts mais complexos, alinhando e distribuindo espaços entre os itens de maneira eficiente.


    • Layout em grade:

  • O modelo de layout flexível (display: flex;) permite a criação de layouts mais complexos, alinhando e distribuindo espaços entre os itens de maneira eficiente.

    Posicionamento:


    O posicionamento em CSS refere-se à técnica utilizada para controlar a localização de elementos na página em relação a outros elementos. Isso é crucial para criar layouts complexos e posicionar elementos de maneira específica.


    Valores da propriedade 'position':

  • static: É o valor padrão. Os elementos são posicionados no fluxo normal do documento.
  • relative: Um elemento é posicionado em relação à sua posição normal.br>
  • absolute: Um elemento é retirado do fluxo normal e posicionado em relação ao ancestral mais próximo que não é static.
  • fixed: Um elemento é retirado do fluxo normal e é posicionado em relação à viewport, permanecendo fixo mesmo durante a rolagem.
  • sticky: Um elemento é tratado como relative até que ele atinja uma determinada posição durante o rolar, momento em que se torna fixed.


    • Propriedades Relacionadas:

  • top, right, bottom, left: Especificam a posição de um elemento em relação ao seu contêiner posicionado.
  • z-index: Controla a ordem de empilhamento dos elementos, determinando qual elemento fica na frente ou atrás de outros.
