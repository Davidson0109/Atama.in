Entenda HTML

O que é:

HTML ou Hyper Text Markup Language é a linguagem de marcação padrão para a construção de de sites Web.

Linguagem de marcação são sinais e códigos que definem o formato e como o texto irá exibir.


Os elementos dentro do código indicarão para o navegador como mostrar o conteúdo. Eles dão significado a tal conteúdo,

como dizer que "isto é um parágrafo", "isto é o titulo" e outras coisas nesse sentido, lembrando que isso sempre eh representado por um sinal dentro do código.


Observe a programação a seguir:

  • O "!DOCTYPE html" é uma declaração que define que o documento de HTML5;

  • O "html" permite que a programação em forma de html se inicie.

  • O "head" serve para colocar metainformações do código, como definir o titulo do próprio.

  • O "title", como o próprio nome ja diz, define o título da página, o que aparecerá na guia.

  • O "body" marca o corpo do documento é um container para todas as coisas vísieis do seu código, como texto, imagens, links entre outras coisas.

  • O "h1" é um elemento que exibe um texto com um tamanho maior.

  • O "p" define um parágrafo no seu documento
